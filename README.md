# Python Case Study — Risk Pricing

## Project Description

This project analyses data on insurance policies between 2018-2020 to investigate the following questions:

- Is the company profitable?
- Which factors affect the frequency and severity of insurance claims?
- Are there any pricing opportunities for customers' insurance plans?

## Folder Contents

This project contains four main directories:

1. Data – stores input and derived datasets, separated into *Raw* and *Clean*.
2. Reports – stores output files (e.g., aggregated datasets).
3. Figures – stores generated figures.
4. Programs - stores additional .py files that assist with data analysis.

## Project Set-up and Execution

The project pipeline is implemented in the Jupyter Notebook `insurance_analysis.ipynb` in the root project directory. The project uses relative file paths, so no modification is required after cloning.

No dataset is included in this repository as all data is owned by Hastings Direct.

## Project Dependencies

This project uses Python libraries for data analysis and visualisation, including:

- pandas – data manipulation
- numpy – numerical operations
- matplotlib and seaborn – data visualisation
- jupyter – interactive notebook environment

All required dependencies are listed in `requirements.txt`.
